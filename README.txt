A maze game developed for A-level NEA project



How do play this game?

1. Download the project as a zip file

2. Decompress the file called "Maze-Game-main"

3. Open python idle or any IDE support python language

4. Press open file or press Ctrl + O

5. Found the decompressed file

6. Select "Maze_Game.py"

7. Press Enter

8. Install pygame

9. Press F5 to execute the code
  
  
  
This game is still in development, sometimes may not work.
